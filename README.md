# the-bad-ball
一个无聊的小游戏，个人Unity初作

简陋，但是加了skybox提升一点点颜值

控制小球，躲避障碍物，并抵达终点

# 编译
Unity版本：2021.3.6f1c1（LTS）

内增超级垃圾代码，逻辑关系混乱，以及乱码的注释

窗口化是因为我自己电脑太烂，只能以降低分辨率的方式提升帧率

别问了，什么问题都是下个版本修（瘫
